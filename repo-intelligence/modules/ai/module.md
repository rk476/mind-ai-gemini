# AI Module

## Purpose
Provides language model logic for reasoning about browser context.

## Primary Responsibilities
- Analyzing DOM nodes to figure out interactable elements.
- Planning tests based on natural language instructions.
- Summarizing final run outcomes.

## Routes or Entry Points
- `lib/ai/domAnalyzer.ts`
- `lib/ai/testPlanner.ts`
- `lib/ai/resultSummarizer.ts`

## Dependencies
- `lib/mcp`

- TestPlanner
