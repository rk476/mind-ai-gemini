# MCP Module

## Purpose
Provides Model Context Protocol structures and primitives for interacting with the browser tools.

## Primary Responsibilities
- Registering tools for AI agents.
- Executing selectors efficiently within the browser context.

## Routes or Entry Points
- `lib/mcp/browserTools.ts`
- `lib/mcp/mcpServer.ts`
- `lib/mcp/selectorEngine.ts`

## Dependencies
- `playwright`

## Key Components
- SelectorEngine
- MCPServer configurations
